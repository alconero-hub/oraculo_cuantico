Este es un bot de trading experimental que utiliza **computación cuántica real** (IBM Quantum) para analizar la volatilidad de Bitcoin y generar veredictos de mercado.

## 🚀 ¿Cómo funciona?
1. **Captura de Datos**: Descarga los últimos 60 minutos de velas de BTC-USD.
2. **ADN Cuántico**: Transforma los cambios de precio en ángulos de rotación.
3. **Procesamiento en QPU**: Ejecuta un circuito de entrelazamiento en un procesador real de IBM.
4. **Veredicto**: Mide el colapso de los cúbits para determinar la tendencia.
